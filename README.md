<h1 align="center">Site Craques do Brasileirão</h1>

![Badge Finalizado](http://img.shields.io/static/v1?label=STATUS&message=FINALIZADO&color=GREEN&style=for-the-badge)

O Craques do Brasileirão é um site moderno e minimalista de mostrando os principais jogadores do Futebol de Brasileiro em **Next.js**, com os dados dos jogadores vindo através de um arquivo JSON.

# Funcionalidades
- **Exibe as informações dos jogadores**
- **Exibe as imagens dos jogadores**
- **Possuí uma Navbar onde pode selecionar o jogador**
- **Site 100% responsivo**

# Tecnologias Utilizadas
- **Next.js 14**
- **React**
- **TypeScript**
- **TailwindCSS**
- **JSON**

# Estrutura do Projeto (src)
- app/ (`globals.css`, `layout.tsx`, `page.tsx`)
- app/players/ (`page.tsx`, `players.module.css`)
- public/ (`players.json`)

# Instalação
- **Passo 1:** Clone o repositório
git clone https://github.com/victorsampaio18/clima-app.git
`cd weather-website`

- **Passo 2:** Instale as dependências
`npm install`

- **Passo 2.1:** Se estiver usando Mac, instale o pods no iOS
`npx pod-install`

- **Passo 3:** Execute o app
`npm run dev`
